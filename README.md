## Instructions

### vimrc

installation script

```bash
curl -fLo ~/.vim/autoload/plug.vim --create-dirs \ && wget https://raw.githubusercontent.com/yashkathe/Linux-Dot-Files-and-Scripts/refs/heads/master/.vimrc
```

open vim and type

```bash
:PlugInstall
```

### bashrc
